Introduction

A Proof is often described as a poem. This originally indicates the cunningness and artistic interactions it often takes to find ways to prove whatever we want to prove. Sadly, we often see proofs written as a poem, be it literally. Vague descriptions of why a certain property is what it is, not seldomly surrounded with intimidations, questioning the intelligence of the reader, are no exception. A complete proof based on axioms only without any shortcuts can rarely be found. These shortcuts hide a lot of assumptions. We often think these assumptions don't hide anything relevant... This kind of proof technique is sometimes used to prove such things as 1 = -1 and other invalid proofs.

If anything, these proofs are a warning, that if we're not careful we may be proving something false. So if poems aren't the way to prove something, what is?

Formal proof

A formal proof, in this context, is to start from a formula or a set of formulas and prove the goal using these formulas and axioms only. Let there be no doubt, this is not an easy task.

How we do a formal proof here

You start out by defining the axiomset you want to use.
What is to be proven we shall now call the goal. You start from no or several formulas that are true. Based on this you can now use axioms from the axiomset to get to the goal.
We apply the formulas of the axioms to the current formula, this is then transformed to a new formula. After a chain of axioms applied to the current formula we transform it to become the goal.

How to protect against cheating

It is important to protect against cheating against the axiomatic assumptions, so it is impossible to prove something that isn't true.(The assumptions must be true.) You start from a formula or multiple formulas that are true. Axioms will always create a new formula that is true. By not allowing the user to add a formula or only under very strict conditions it is impossible make a false proof. The problem is that the axioms must therefore only generate a proof that is true. If one of the assumptions in the axioms is wrong we can prove that true is false and therefore we can prove anything.
